#### 理论SSIM公式直接计算完整参考图像与失真图像之间的全局结构相似性。但观察官方源码发现，SSIM的是基于滑动窗计算局部SSIM取平均得到的，请问为什么要这么做（5分）

1. **局部信息更能反映人眼视觉特性**：人类视觉系统更敏感于局部的细节变化，而不仅仅是图像整体的变化。滑动窗口的局部 SSIM 计算能够捕捉到这些细节，特别是那些可能在全局计算中被忽略的局部对比度和结构的变化。

2. **适应图像的非均匀性**：图像在不同区域可能会有显著差异。直接计算全局 SSIM 会导致局部失真被平均化，无法反映失真图像中局部区域的质量变化。局部 SSIM 能更好地捕捉这些区域差异，使得计算更加符合图像内容的真实分布。

3. **提高对各种失真的敏感性**：不同类型的失真对图像不同区域的影响可能差异很大。局部 SSIM 的方法能够捕捉不同区域的失真影响，例如某些区域的模糊或噪声，会显著影响局部 SSIM 值，使得 SSIM 评价能更准确地反映出图像质量。

4. **对边缘、纹理和细节的准确衡量**：图像的边缘、纹理和细节区域往往在质量评价中至关重要。滑动窗口计算局部 SSIM，使得这些关键区域的对比度和结构变化能够更精确地衡量，不会因全局计算而被“平滑”或忽略。
