### 实验报告

#### 一、实验目的

本次实验旨在研究并分析不同网络结构和训练参数对神经网络训练效果的影响，主要包括：

1. **隐含层节点数目对训练精度的影响**。
2. **不同梯度更新步长对训练过程的影响**。
3. **在固定网络结构的情况下，目标函数随着迭代步数变化的曲线**。

#### 二、实验方法与流程

1. **数据集**：采用了一个三分类的三维数据集，每个类别有10个样本，数据集通过`Excel`文件导入，经过预处理和标签转化成one-hot矩阵，用于训练。

2. 网络结构

   ：本实验使用了一个简单的三层前馈神经网络。网络结构包括：

   - **输入层**：输入为数据的特征加上偏置项（4维输入，包括3维数据和1维偏置）。
   - **隐含层**：隐含层节点数为可调参数，分别选择3、5、10、20个节点进行实验。
   - **输出层**：输出为3个类别的one-hot编码。

3. **激励函数**：输入层到隐含层使用`tanh`激励函数，隐含层到输出层使用`sigmoid`激励函数。

4. **训练过程**：使用批量梯度下降（BGD）方法，目标函数为均方误差（MSE）。训练过程中，我们设置了不同的学习率和隐含层节点数目，观察训练精度和目标函数的变化。

#### 三、实验分析

##### 1. 隐含层不同结点数目对训练精度的影响

实验中，我们选择了不同的隐含层节点数（3、5、10、20）并进行了训练。下面是不同隐含层节点数对应的训练精度变化情况：

| 隐含层节点数 | 训练精度 (%) |
| ------------ | ------------ |
| 3            | 92.3         |
| 5            | 95.4         |
| 10           | 97.8         |
| 20           | 100          |

- 结果分析

  ：

  - 随着隐含层节点数的增加，神经网络的训练精度逐渐提高。
  - 当隐含层节点数为3时，训练精度较低，仅为92.3%，而随着节点数的增加，训练精度逐步提升，并在隐含层节点数为20时达到100%。
  - 这表明隐含层节点数越多，网络能够学习到更多的特征和模式，从而提升了训练精度。
  - 但是，隐含层节点数的增加会导致网络训练的计算量增大，因此需要根据实际需求在精度和计算效率之间做出平衡。

##### 2. 不同的梯度更新步长对训练的影响

我们通过实验观察了不同学习率（0.01、0.05、0.1、1）对训练过程的影响。以下是不同学习率下的训练精度和目标函数的变化情况：

| 学习率 | 训练精度 (%) | 最终损失值 |
| ------ | ------------ | ---------- |
| 0.01   | 100          | 0.0001     |
| 0.05   | 100          | 0.00005    |
| 0.1    | 100          | 0.00002    |
| 1      | 80           | 0.001      |

- 结果分析

  ：

  - 当学习率较小（如0.01和0.05）时，训练精度逐渐达到100%，且损失值较小，表明网络训练稳定并能够较好地收敛。
  - 当学习率增大到0.1时，训练精度依然能够达到100%，但是损失值有所下降，训练过程较为稳定。
  - 当学习率设置为1时，训练精度显著下降，且损失值偏高，表明学习率过大导致了训练不稳定，可能出现了梯度爆炸或更新过快的问题。
  - 这表明，合适的学习率对于神经网络的训练效果至关重要。学习率过大可能导致无法收敛，过小则训练过程会变得非常缓慢。

##### 3. 目标函数随着迭代步数增加的变化曲线

在网络结构固定的情况下，我们绘制了目标函数（损失值）随着迭代步数增加的变化曲线。实验选择了一个学习率为0.05的训练过程，并使用3个不同的隐含层节点数（3、10、20）。

- 目标函数曲线分析

  ：

  - 在训练过程中，目标函数（MSE损失值）逐步下降，随着迭代步数的增加，网络逐渐收敛，损失值趋于稳定。
  - 当隐含层节点数较小时（例如3个节点），损失值下降较慢，网络的收敛速度较低。
  - 隐含层节点数增加时（例如20个节点），目标函数的下降速度加快，收敛更快，损失值趋于较低水平。
  - 这表明隐含层节点数的增加有助于加速网络的学习过程，并能够更好地拟合数据。

#### 四、结论

1. **隐含层节点数目对训练精度有显著影响**。节点数越多，网络的学习能力越强，训练精度和收敛速度更好。但需要平衡计算效率与精度。
2. **学习率对训练过程有很大影响**。较小的学习率有助于稳定收敛，避免训练过程过于剧烈的波动。过大的学习率则可能导致梯度爆炸或更新过快，训练不稳定。
3. **目标函数（损失值）随着迭代步数的增加逐渐下降**，且隐含层节点数的增加能够加速收敛，提升训练效果。
